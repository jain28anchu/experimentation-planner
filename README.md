# Experimentation Planner

AI-powered experimentation and A/B testing planning tool for product managers.

## Features

- Executive experiment summaries
- Experiment quality scoring
- AI-generated recommendations
- Rollout planning
- Guardrail metric recommendations
- PM leadership questions
- Experiment timelines
- Risk analysis
- Downloadable experiment plans
